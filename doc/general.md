<h2> #character </h2>

```c
title:[string]
// set the title of the website(<title> tag) (not required)

dialogBox:[path]
// set the image of your dialogBox (not required)

characterCount:[uint]
// set the amounts of your characters (not required)
```

<h3>Example Code</h3>

```c
#general
title:"Hello World" // set the title to Hello World
dialogBox:"images/example.png" // set the dialogBox image to example.png
characterCount:12   // the whole script will have 12 characters
```
